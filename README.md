# Tianle Cao, Ph.D. - Personal Website

This is my personal professional portfolio and research blog, built with [Astro](https://astro.build).

## Project Structure

- `src/content/blog/`: Learning posts and technical articles.
- `src/content/projects/`: Research projects and publications.
- `src/content/work/`: Professional experience history.
- `src/content/honors/`: Academic and professional awards.
- `src/content/education/`: Academic background.
- `public/`: Static assets like my CV, resume, and images.

## Local Development

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Deployment

Automated via GitHub Actions to GitHub Pages on every push to the `main` branch.
